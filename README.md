## Research Focus

I build AI systems that combine neural models and symbolic reasoning to improve reliability, interpretability, and adaptability in decision-making workflows. Current work includes fast-slow architectures for reasoning, knowledge-grounded planning, reinforcement learning with editable specifications, and heuristic learning for search.
